# Component Documentation

## UI Components
- `Button`: Standardized button with variants (primary, secondary, danger, ghost, outline) and loading states.
- `Input`: Form input with label, error message, and left icon support.
- `Badge`: Status and type indicator.
- `Modal`: Accessible modal dialog with backdrop blur.
- `Spinner`: Loading indicator.
- `EmptyState`: Placeholder for empty lists/data.
- `ConfirmDialog`: Reusable confirmation modal for destructive actions.

## Poll Components
- `PollCard`: Summary card for feeds and lists.
- `PollForm`: Form for creating and editing polls (with Zod validation).
- `OptionEditor`: Dynamic list editor for poll options.
- `ResultBars`: Animated progress bars for poll results.

## Voting Components
- `VoteForm`: Checkboxes (multi-choice) or radio buttons (single-choice) for voting.
- `VoteStatus`: Displays the user's current vote and allows withdrawal.

## Invitation Components
- `InvitationManager`: UI for inviting users via email and managing active/revoked invitations.
